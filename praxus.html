¿Qué es Praxus?

Un sistema web que ofrece una solución integral a las organizaciones educativas para administrar y verificar prácticas profesionales de forma eficaz y eficiente.

  1. Inicio
  2. ¿Qué es Praxus?

Problemas Identificados y soluciones

Gran cantidad de practicas

Las organizaciones educacionales deben revisar un gran volumen de prácticas, estas pueden incluir múltiples documentos o informes, este proceso puede ser lento y tedioso si no se lleva un orden adecuado o no se tienen herramientas para facilitar la revisión. Para esto, Praxus se plantea como un sistema:

  • Adaptable a las necesidades de las organizaciones educativas
  • Fácil de usar para todos los agentes involucrados
  • Que ayuda a ordenar el trabajo del encargado de revisar practicas

Asegurar veracidad de practicas

Para las organizaciones educacionales es necesario asegurar la veracidad de la práctica que realiza un estudiante , de esta forma, es posible confirmar el correcto desarrollo de un profesional en formación. Para esto, Praxus cuenta con:

  • Verificación de datos de supervisor y empresa donde el estudiante realiza su practica
  • Análisis de consistencia entre evaluación e informe del supervisor empleando IA
  • Análisis de consistencia entre informes del estudiante y supervisor empleando IA
